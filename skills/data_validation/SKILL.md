# Data Validation Skill

## Scopo

Validare dataset caricati da CSV, Excel o Oracle prima dell'elaborazione.

## Controlli richiesti

- Verificare presenza di righe e colonne.
- Evidenziare valori mancanti per colonna.
- Identificare righe duplicate.
- Segnalare tipi dati inattesi.
- Distinguere problemi bloccanti da avvisi.
- Non inventare risultati: usare solo dati presenti nel contesto.

## Output atteso

Rispondere sempre in italiano con:

- esito della validazione;
- problemi trovati;
- impatto sull'analisi;
- raccomandazioni operative.
