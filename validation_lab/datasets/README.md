# Datasets

Contiene dataset piccoli o riferimenti ai dataset usati nei test del Validation Lab.

Regole:

- Non salvare dati sensibili o personali reali.
- Preferire dataset sintetici o anonimizzati.
- Documentare schema, dominio e origine nel test case collegato.
- Evitare file grandi: usare sample ridotti e riproducibili.
