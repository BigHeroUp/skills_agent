# Architecture Reviews

Contiene review architetturali nate da problemi ripetuti o decisioni tecniche rilevanti.

Usare questa cartella quando:

- un bug richiede una decisione cross-module;
- una fix locale rischia di introdurre debito;
- serve confrontare alternative;
- un quality gate fallisce ripetutamente.

Usare `../architecture_review_template.md` come base.
