# Expected Results

Contiene risultati attesi dettagliati per test case complessi.

Usare questa cartella quando il campo `Expected Result` del test case diventa troppo lungo.

Ogni file dovrebbe indicare:

- test case collegato;
- KPI attesi;
- grafici attesi;
- esclusioni attese;
- comportamento follow-up;
- quality gate rilevanti.
