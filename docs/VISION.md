# Skills Agent V2 Vision

## Thesis

Skills Agent V2 formalizza il progetto come una **Analytical Intelligence Platform**
offline-first, progettata per ragionare sui dati prima di descriverli.

La piattaforma deve comportarsi come un **Senior Data Analyst digitale**:

- interpreta l'obiettivo analitico;
- struttura un piano di analisi verificabile;
- seleziona metriche, controlli e confronti rilevanti;
- riusa esperienza analitica accumulata nel tempo;
- produce raccomandazioni motivate e auditabili.

## Product Identity

Skills Agent V2 non e un chatbot generalista e non e un semplice wrapper di LLM.
E una piattaforma di intelligence analitica che usa:

- dati reali;
- modelli statistici espliciti;
- memoria analitica persistente;
- reasoning deterministico;
- componenti linguistici opzionali.

## Strategic Goal

Il core decisionale deve funzionare anche senza API OpenAI o altri LLM remoti.

Il valore principale della V2 deriva da:

- motore di pianificazione analitica;
- motore statistico;
- Knowledge Graph locale;
- reasoning engine esperienziale;
- futuri motori di esperienza e raccomandazione.

Gli LLM possono arricchire la forma narrativa, ma non devono essere il motore
decisionale primario.

## Long-Term Direction

La piattaforma deve accumulare esperienza analitica in modo progressivo:

- quali metriche sono state usate;
- quali anomalie sono ricorrenti;
- quali root cause sono gia state osservate;
- quali strategie hanno funzionato in casi simili;
- quali report e quali spiegazioni sono risultate piu utili.

Questo rende Skills Agent V2 una piattaforma capace di apprendere esperienza
analitica mantenendo risultati riproducibili, spiegabili e offline-first.

## Non-Negotiable Principles

- Il reasoning core non deve dipendere da LLM.
- Nessuna decisione analitica critica deve dipendere esclusivamente da output narrativi.
- I risultati numerici devono essere deterministici o statisticamente espliciti.
- La memoria analitica deve salvare metadati e sintesi, non righe raw dei dataframe.
- L'architettura deve restare modulare e pronta a crescere per layer.
