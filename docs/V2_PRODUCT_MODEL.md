# Skills Agent V2 Product Model

## Product Positioning

Skills Agent V2 e una piattaforma offline-first di Analytical Intelligence
pensata per aiutare professionisti tecnici a trasformare dati e richieste di
business in analisi affidabili, spiegabili e progressive nel tempo.

## Utente Target

- data analyst
- developer
- technical project manager
- business analyst tecnico

## User Value

La piattaforma offre:

- un Senior Data Analyst digitale;
- reasoning locale e auditabile;
- riuso di esperienze analitiche precedenti;
- suggerimenti analitici concreti;
- report professionali anche senza LLM.

## Core Use Cases

- analisi CSV/Excel
- analisi Oracle
- anomaly detection
- root cause analysis
- confronto analisi
- riuso esperienze precedenti
- suggerimenti analitici
- report professionali

## Operating Model

### Input

- file CSV;
- file Excel;
- query Oracle read-only;
- richieste utente in linguaggio naturale;
- metadata di contesto.

### Processing

- profiling dataset;
- planning analitico;
- esecuzione statistica;
- rilevazione anomalie;
- root cause analysis;
- reasoning su memoria analitica;
- composizione del report.

### Output

- KPI e trend;
- insight deterministici;
- anomalie e cause possibili;
- raccomandazioni analitiche;
- report professionale;
- spiegazioni opzionali in linguaggio naturale.

## Product Boundaries

Skills Agent V2 non deve:

- dipendere obbligatoriamente da OpenAI;
- sostituire il calcolo con testo narrativo;
- salvare dati raw sensibili nel layer di memoria analitica;
- prendere decisioni critiche solo su base linguistica.

## Future Expansion

Il modello prodotto e pensato per crescere verso:

- marketplace di Domain Pack;
- Experience Engine;
- Recommendation Engine;
- Decision Intelligence Layer;
- narrative layer LLM opzionale.
